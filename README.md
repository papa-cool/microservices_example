## Create executable file

Display `hello world` with an executable ruby file.

<details>
  <summary>Solution</summary>

```sh
mkdir bin
touch bin/console.rb
```

In `bin/console.rb`.
```ruby
#!/usr/bin/env ruby

puts 'Hello World'

```
